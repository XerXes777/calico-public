Microcontroller Specifications

Technische Daten des benötigten ESP32-S3-Moduls.

Benötigter Mikrocontroller:

  • Board-Bestellcode: ESP32-S3-WROOM-1-N16R8
  • ROM: 384 KB
  • SRAM: 512 KB
  • RTC SRAM: 16 KB
  • Flash Größe: 16 MB (Quad SPI)
  • PSRAM Größe: 8 MB (Octal SPI)
