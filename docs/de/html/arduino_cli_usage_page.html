Arduino CLI Usage

Anleitung zum Kompilieren und Hochladen mit Arduino CLI.

Hinweis für Arduino-IDE-Nutzer

Wer lieber eine grafische Oberfläche nutzt, kann weiterhin die Arduino IDE verwenden.
Diese Anleitung richtet sich an Fortgeschrittene, die ohne Editor oder automatisiert arbeiten möchten.

Einstieg mit der Arduino CLI

  1. Projekt herunterladen: Entpacke das Archiv nach:
    C:\Users\<DeinBenutzername>\Documents\Arduino
  2. Terminal öffnen:
    • Win + R, cmd eingeben und Enter.
    • Zum Projektordner navigieren:
      cd "%USERPROFILE%\Documents\Arduino"
  3. Arduino CLI konfigurieren:
    arduino-cli config init --overwrite
    arduino-cli config set directories.data "%USERPROFILE%\Documents\Arduino\platforms"
    arduino-cli config set directories.downloads "%USERPROFILE%\Documents\Arduino\platforms\temp"
    arduino-cli config set directories.user "%USERPROFILE%\Documents\Arduino"
    arduino-cli config set directories.builtin.libraries "%USERPROFILE%\Documents\Arduino\platforms\libraries"
  4. Konfiguration prüfen:
    arduino-cli config dump
  5. Core-Index aktualisieren:
    arduino-cli core update-index

  6. Angeschlossene Boards anzeigen:

    arduino-cli board list

    Beispielausgabe:

    Port Protokoll Typ Platinenname FQBN Kern
    COM5 serial Serial Port (USB) Unbekannt
  7. Projekt kompilieren:
    arduino-cli compile YourProjectName --clean --log-level warn --log --build-path "%USERPROFILE%\Documents\Arduino\YourProjectName\build"
  8. Firmware hochladen:
    arduino-cli upload YourProjectName -p COM5 --log-level warn --log --input-dir "%USERPROFILE%\Documents\Arduino\YourProjectName\build"
  9. Seriellen Monitor öffnen:
    arduino-cli monitor YourProjectName -p COM5 --config 115200

  10. Upload und Monitor kombinieren:

    arduino-cli compile YourProjectName --clean --log-level warn --log --build-path "%USERPROFILE%\Documents\Arduino\YourProjectName\build" & arduino-cli upload YourProjectName -p COM5 --log-level warn --log --input-dir "%USERPROFILE%\Documents\Arduino\YourProjectName\build"

    Mit Strg + C beenden.

Hinweise

  • Stelle sicher, dass das Board angeschlossen ist und Treiber installiert sind.
  • Ersetze COM5 mit dem passenden Anschluss.
  • Die CLI-Konfiguration muss nur einmal durchgeführt werden.
