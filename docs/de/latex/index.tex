Dokumentation zum Gerät.  \hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Einführung}{Einführung}}\label{index_intro_sec}
Dieses Gerät trocknet Filament und hält es auf optimaler Temperatur, um die Qualität von 3D-\/\+Drucken zu verbessern.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Funktionen}{Funktionen}}\label{index_features_sec}

\begin{DoxyItemize}
\item {\bfseries{Einfache Bedienung}}\+: Steuerung über einen Drehencoder.
\item {\bfseries{Echtzeit-\/\+Anzeige}}\+: Temperatur und Luftfeuchtigkeit auf einem LCD.
\item {\bfseries{Vordefinierte Presets}}\+: Optimierte Profile für PLA, PETG, ABS und mehr.
\item {\bfseries{Effiziente PID-\/\+Regelung}}\+: Reaktionsschnell und energiesparend.
\item {\bfseries{Flexible Integration}}\+: Potentialfreie Start/\+Stopp-\/\+Kontakte (3.\+3V–24V kompatibel).
\item {\bfseries{Status-\/\+LEDs}}\+: Anzeigen für Heizung, Lüfter und Gesamtsystem.
\item {\bfseries{Zwei Modi}}\+: Steuerung über Temperatur oder Luftfeuchtigkeit.
