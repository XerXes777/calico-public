This device is designed to enhance your 3D printing experience by maintaining optimal filament conditions. This guide provides step-\/by-\/step instructions for using the system effectively.\hypertarget{user_guide_page_autotoc_md19}{}\doxysubsubsection{\texorpdfstring{Getting Started}{Getting Started}}\label{user_guide_page_autotoc_md19}

\begin{DoxyEnumerate}
\item {\bfseries{Powering On\+:}}
\begin{DoxyItemize}
\item Connect the device to a power source.
\item Upon startup, it will initialize and show current temperature and humidity on the display.
\end{DoxyItemize}
\item {\bfseries{Selecting Filament Presets\+:}}
\begin{DoxyItemize}
\item Use the rotary encoder to browse through presets for common filament types (PLA, PETG, ABS, etc.).
\item Press the encoder to select one. The device automatically adjusts to ideal settings.
\end{DoxyItemize}
\item {\bfseries{Manual Adjustment\+:}}
\begin{DoxyItemize}
\item You can manually fine-\/tune temperature and humidity using the encoder.
\item Long-\/press to switch between modes, then rotate to adjust values.
\end{DoxyItemize}
\item {\bfseries{Operation\+:}}
\begin{DoxyItemize}
\item Press "{}\+Start"{} to activate heating and airflow.
\item Press "{}\+Stop"{} to shut down the system.
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{user_guide_page_autotoc_md20}{}\doxysubsubsection{\texorpdfstring{Indicators}{Indicators}}\label{user_guide_page_autotoc_md20}
LED indicators give quick status feedback\+:
\begin{DoxyItemize}
\item {\bfseries{Heating LED}} – active when the heater is on
\item {\bfseries{Fan LED}} – active when the fan is running
\item {\bfseries{System LED}} – general on/off indication
\end{DoxyItemize}\hypertarget{user_guide_page_autotoc_md21}{}\doxysubsubsection{\texorpdfstring{Maintenance}{Maintenance}}\label{user_guide_page_autotoc_md21}
For best performance\+:
\begin{DoxyItemize}
\item Clean fan and heater to avoid dust buildup.
\item Check wiring and connectors regularly.
\item Replace desiccant in the filament container as needed.
\end{DoxyItemize}\hypertarget{user_guide_page_autotoc_md22}{}\doxysubsubsection{\texorpdfstring{Troubleshooting}{Troubleshooting}}\label{user_guide_page_autotoc_md22}

\begin{DoxyItemize}
\item {\bfseries{No power\+:}} Verify the power source and cable.
\item {\bfseries{No change in values\+:}} Ensure sensors are properly connected.
\item {\bfseries{No LED activity\+:}} Check all wiring and components.
\end{DoxyItemize}
