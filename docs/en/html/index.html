Introduction

This device dries filament and maintains optimal filament temperature to enhance 3D print quality.

Features

  • Easy Operation: Intuitive control via rotary encoder.
  • Real-Time Display: Continuous temperature and humidity shown on an LCD.
  • Predefined Presets: Ready-to-use profiles for PLA, PETG, ABS, and more.
  • Efficient PID Control: Responsive, stable, and energy-saving heater management.
  • Flexible Integration: Potential-free start/stop inputs compatible with 3.3Vâ€“24V systems.
  • Status Indicators: LEDs show heater, fan, and system states.
  • Dual Modes: Operate based on temperature or humidity control.
